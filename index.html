Save Button

Cancel Button

  • Exits the Options Screen and the Option Sets Screen is displayed

    • Note: If there are any unsaved changes to the currently selected record, the tool will prompt to save

  • Creates a new record in the database (add)

    • Note: If edits are made without clicking add first, you will be updating an existing record in the database (update)

  • Deletes the currently selected record from the database (delete)
  • Presents a confirmation prompt

    • Note: This action cannot be undone

  • Saves any changes made to the selected record (update), or
  • Saves any changes made to a new record (add)
  • Cancels any changes made to the selected record (update), or
  • Cancels any changes made to a new record (add)
This button is avaiable in the following option sets:
  • Item Identify List
  • Item Identify Data
  • Store Configurations
  • Boss Configurations
  • Consolidated Item Security
